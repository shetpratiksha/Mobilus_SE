/////////////////// Layout : 1 Column, 2 Columns (left column), (don't use 3 columns)

///////////////////Product Images (bestsellers, specials, latest, featured)

--- Default size : 80x80

--- Modify size 40 x 40 on the side
- Admin Cpanel > Extensions > Modules > Featured (Bestsellers, Latest, Specials)
- Image(W x H) : 40 x 40
- Position : Column Left (or Column Right)
- Save


--- Modify size 160x160 at the content
- Admin Cpanel > Extensions > Modules > Featured (Bestsellers, Latest, Specials)
- Image(W x H) : 160 x 160
- Position : Content Top (or Content Bottom)
- Save


--- Modify size "Related Product"
- Admin Cpanel > System > Settings > Image
- Related Product Image Size : 160 x 160
- Save
